# Landing Page Sessions

## Purpose
Daily working notes for landing page development - separate from platform sessions.

## Structure
```
sessions/
├── current/     # Active work
└── archive/     # Older than 7 days
```

## Rules
1. **One file per day**: YYYY-MM-DD.md
2. **Landing page only**: Platform work goes in platform repo
3. **Archive weekly**: Move old files to archive/
4. **Reference PENDL_LANDING.md**: For strategic context

## Template
```markdown
# Session: [Date]

## Goal
[From landing page PRIORITIES.md]

## Notes
[Landing page specific work]

## Completed
[Update landing page CHANGELOG.md]
```
